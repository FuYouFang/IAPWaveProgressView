## Example

clone 代码后，在运行 demo 前，首先在 Example 目录下运行 `pod install` 命令。
## Requirements

- iOS 9.0 以上
## Installation

通过 pod 使用 IAPWaveProgressView：

```ruby
pod 'IAPWaveProgressView'
```
